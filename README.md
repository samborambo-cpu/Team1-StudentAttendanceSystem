# Student Attendance System

## Team Members
- Team Member Samuel Azize
- Team Member Luis Manslave
- Team Member Camilo Sequeira
  

## Project Description
A system designed to track student attendance for a class. The application manages attendance records for up to 25 students and generates attendance reports.

## Project Goals
- Add attendance records
- Search student records
- Display attendance information
- Update attendance status
- Generate attendance summary reports
